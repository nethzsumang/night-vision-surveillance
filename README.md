### Packages Required:
* opencv-python (Object Detection and Video tools)
* imutils (Video streaming)
* var_dump (Debugging)
* numpy (Matrix Computations)
* matplotlib (Image display)
* pygame (for audio alert)

### Python Version Needed: 
* Python 3.6.x and up
